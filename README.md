# 📦 Pre-Sales SSI Pack — Microsoft 365 / Security / Backup

This repository illustrates **how IT pre-sales work is evolving** with the help of AI.

Following the publication *“Something Big Is Happening”* by **Matt Shumer (February 2026)**, I wanted to test **what AI can actually deliver**, in a **realistic IT pre-sales / SSI context**.

👉 Goal: demonstrate that a **structured, coherent, client-ready pre-sales package** can now be **pre-built in under one hour**, without replacing human expertise.

---

## 📂 Repository content

- 📐 **Microsoft 365 Architecture** *(PowerPoint)*  
  Target architecture for a Swiss SMB (15 users):  
  Microsoft 365 Business Premium, security, Swiss **nLPD compliance**, Veeam backup  
  (local disk + LTO GFS + immutable Azure copy).

- 🛠️ **Technical Procedure** *(Word)*  
  Step-by-step technical documentation, intentionally written for a **junior technician**  
  (tenant setup, security, Intune, Purview, Veeam, operations).

- 👥 **User Guide** *(Word)*  
  Simple and pedagogical end-user guide: Outlook, OneDrive, Teams, SharePoint, MFA,  
  and security best practices.

---

## ⏱️ Method

    Iterative Prompting Strategy: The package was built using 3 distinct core prompts (Architecture, Technical Procedure, and User Guide) to ensure depth and precision for each deliverable.

    Expert Refinement: I performed 3 to 4 manual iterations specifically on the Technical Procedure to ensure the M365 and Veeam configurations were 100% accurate and client-ready.

    Production Time: ~2 hours of total work, including AI steering, technical validation, and final formatting (a task that usually takes several days of manual drafting).

    Human-in-the-loop: The AI acted as a high-speed architect, while the Pre-Sales Engineer remained responsible for technical consistency and final pricing validation.

👉 AI structures and calculates — **pricing responsibility remains human**.
